![SparkFun RJ45 Breakout Pin Numbers](doc/images/SparkFunRj45Breakout.jpg?raw=true "SparkFun RJ45 Breakout Pin Numbers")

| Pin Number    | Function      |
| ------------- |:-------------:|
| 1             | RS485 TX+ / Y |
| 2             | +9VDC         |
| 3             | TS485 RX+ / A |
| 4             | GND           |
| 5             | +9VDC         |
| 6             | GND           |
| 7             | RS485 RX- / B |
| 8             | RS485 TX- / Z |
